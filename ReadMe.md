# Sign In and Sign Up System (C Language)

This project is a simple sign-in and sign-up system built using the C programming language.

## Features:
- Register a new user (Sign Up)
- Login with existing credentials (Sign In)
- Saves data in users.txt

## How to Run:
1. Compile the code: gcc main.c -o app
2. Run the app: ./app (Linux) or app.exe (Windows
